🎓 College Analytics: Exploring ROI, Placements, and Trends with Data-Driven Insights
📌 Project Overview

This project focuses on analyzing 1500+ Indian engineering colleges to evaluate the Return on Investment (ROI) by comparing tuition fees with average placement packages. The goal is to uncover key trends in placements, salaries, and admission exams, helping students and parents make informed decisions when selecting colleges.

🚀 Key Features

📊 ROI Analysis – Compared tuition fees with average placement packages.

📈 Placement & Salary Trends – Identified high-performing colleges and branches.

📝 Admission Exam Insights – Explored the impact of entrance exams on placements.

🎨 Interactive Dashboards – Built with Power BI for dynamic exploration.

🐍 Python Visualizations – Using Pandas, Matplotlib, and Seaborn for deep analysis.

🛠️ Tools & Technologies

Data collection using Web scapeing 

Power BI – Dashboard creation & interactive analysis

Python (Pandas, Matplotlib, Seaborn) – Data analysis & visualization

CSV – Data collection & preprocessing
